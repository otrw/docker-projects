# Docker Learning Projects

## About
A collection of Docker-based setups, documenting my learning toward modern cloud-native tooling by building small hands-on projects. Built to learn by doing, improve repeatability, and develop real operational habits.

> This is a learning repo. Some projects are polished, some are experiments, and all of them reflect progress.
> Mistakes, rewrites, and simplifications are part of the process.

## Current Focus
- Docker fundamentals
- Linux workflows
- Git hygiene
- Practical infrastructure habits

## Projects

| Project | Description |
